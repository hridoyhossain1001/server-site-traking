FAQ

Got questions about Buykori? We've got answers.

Simple answers for campaign tracking, one-page landing pages and event quality.

What is Buykori AdSync?

A server-side tracking platform that sends cleaner WooCommerce events to Meta CAPI, TikTok Events API and GA4.

Does it support one-page landing pages?

Yes. One-page mode can wait for real checkout intent before sending InitiateCheckout.

Can I use only TikTok or only Meta?

Yes. Platform toggles can control which destination receives events for each store.

What data improves event match quality?

Content IDs, value, currency, click IDs, user agent, IP and hashed customer fields help platforms match events better.
