WordPress + WooCommerce Setup
✅
Recommended: Use the official Buykori AdSync WordPress plugin from the plugin download. It sends signed requests, receives updates, and avoids duplicate WooCommerce events.
⚠️
Do not add extra WooCommerce/PHP snippets beside the official plugin. Manual snippets can create duplicate Purchase/AddToCart/ViewContent events.
Step 1: Download the official plugin from the client portal or plugin download endpoint.
Step 2: WordPress Admin → Plugins → Add New → Upload Plugin, then activate it.
Step 3: Open the Buykori AdSync menu, paste the client's API key, and save.
Non-WooCommerce/custom site only: add this site JS:
