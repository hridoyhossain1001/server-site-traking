৫. Must-follow rules

    
    
      
  • event_id unique হতে হবে। Purchase হলে order_1001 type ID ব্যবহার করুন।
    • 
      
  • content_ids Facebook/TikTok catalog product ID-এর সাথে exact match করতে হবে।
    • 
      
  • currency ISO code হবে, যেমন BDT, USD।
    • 
      
  • client_ip_address এবং client_user_agent দিলে match quality ভালো হয়।
    • 
      
  • Email/phone raw দিলেও server auto SHA-256 hash করবে। আগে hash করা থাকলেও accept করবে।
    • 
      
  • একই event browser এবং server দুদিক থেকে পাঠালে একই event_id দিন, না হলে duplicate হতে পারে।
